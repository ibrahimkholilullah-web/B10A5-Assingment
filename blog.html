Whats is the Object Model (DOM)?

The DOM allows web pages to be interactive and dynamic. Developers can change content, styles, and even structure without reloading the page.

How do you select on from the DOM?

To select elements from the DOM (Document Object Model) in JavaScript, there are several methods provided by the document object. These methods allow you to access elements based on their ID, class, tag name, or even complex CSS selectors.

What is event deiegation in the DOM , and why is usful?

Event delegation in the DOM is a technique where a single event listener is attached to a parent element, and it handles events triggered by its child elements.
